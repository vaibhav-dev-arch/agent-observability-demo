Agent Observability Demo v0.1

Utility CS agent simulation — traces, policy boundaries, eval gate. No API keys required.

Start here: Open API docs (/docs)

In Swagger UI, try:

  • POST /api/v1/agents/run — run the stub agent
  • GET /api/v1/agents/traces/{trace_id} — inspect the trace
  • POST /api/v1/evals/run — run the 5-check eval gate

This is not ai-bcdr-governance — paths like /simulate and /evidence are not on this service.
